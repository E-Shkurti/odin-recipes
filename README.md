# odin-recipes

This is a simple HTML project that displays three different recipes.

Description

Each recipe page includes:

- An image of the dish
- A short description
- A list of ingredients
- Step-by-step instructions

Recipes Included:

- Spaghetti Carbonara
- Pizza Margherita
- Pancakes

Purpose

This project was created to practice basic HTML structure, including headings, images, lists, and page organization.
